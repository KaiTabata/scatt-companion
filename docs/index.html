観察できる指標

SCATT 互換指標

10a / 10a-0.5 / 10b / 9c / S1 を本家と同じ命名で。

本家にない補助指標

重心 R95、撃発時速度、力み (8-12Hz)、フォロースルー一貫性。

z-score ランキング

過去 shot との偏差を悪い順に並べる。

手のふるえと呼吸

pre-trigger を周波数別に分解 (呼吸 0.15-0.5Hz / 手ふるえ 8-12Hz)。

反動の詳細

peak 振幅、settle time、フォロースルー R95、反動方向の円形 σ。

心拍 / HRV

Apple Watch (HeartCast) や Polar H10 等を BLE で受信。

自然言語フィードバック

LLM なしで動作。所見をローカルで生成。

セッション横断

前半 vs 後半、ベスト vs ワースト、Series 10 発ブロック表示。

ローカル完結

クラウド送信なし、API キー不要。

3 つのモード

50m スポーツライフル

伏射 (Prone)

SCATT 互換 + 反動詳細 + 心拍。

立射

AR (10m エアライフル)

撃発時速度・S1・フォロースルーの一貫性。

ターゲット非依存

ホールド練習

銃の静止度だけを見る。重心 R95 と力み中心。

入手

前提: SCATT Companion は SCATT Electronics 社の射撃トレーニングソフト SCATT Expert が出力するデータを読みに行く補助ツールです。先に本体をインストールしてください。

SCATT Expert を公式サイトから入手 ↗ scatt.com

OS を選んでください。未署名配布のため、初回のみ簡単な手順があります。

macOS へ Windows へ

macOS 11+

Mac (DMG)

約 260 MB / Apple Silicon & Intel

初回起動 (5 ステップ)

  1. 01

    DMG をダウンロード

    ダウンロード v0.4.12

    約 260 MB / GitHub Releases から配信

  2. 02

    DMG を開いてアプリをドラッグ

    ダウンロードした DMG をダブルクリックすると、ウィンドウが開きます。
    SCATT Companion.app を アプリケーション フォルダにドラッグ & ドロップしてください。

    ※ DMG が開けないと出たら、ファイルを右クリック → 開く。

  3. 03

    ターミナル.app を開く

    キーボードで ⌘ + Space を押すと検索バーが出ます。
    「ターミナル」と入力 → Enter で起動。

    ※ 黒っぽい画面のシンプルなウィンドウが開けば OK。

  4. 04

    下のコマンドをコピーして、ターミナルに貼り付け → Enter

    $ xattr -dr com.apple.quarantine "/Applications/SCATT Companion.app"

    右上の Copy を押すとコピーできます。ターミナルで ⌘ + V で貼り付け、Enter。何もメッセージが出ずに次の行が戻れば成功です。

  5. 05

    SCATT Companion を起動

    Launchpad、または アプリケーション フォルダから SCATT Companion をダブルクリック。
    「インターネットからダウンロードされたアプリです。開いてもよろしいですか？」と表示されたら 開く。

    ※ 2 回目以降はそのままダブルクリックで起動します。

Apple Developer 署名なし配布のため、Gatekeeper の隔離属性を一度だけ解除する手順です。アプリは手元の Mac で完結し、外部に通信しません。

削除macOS から SCATT Companion を消す +

アプリだけ削除する

アプリケーション フォルダから SCATT Companion.app をゴミ箱にドラッグ → ゴミ箱を空にする。

※ 心拍データや設定は残ります。再インストール時に引き継げます。

設定・心拍データも含めて完全に削除する

ターミナル.app で以下を実行 (上のステップ 03 にターミナルの開き方があります)。

$ rm -rf "/Applications/SCATT Companion.app" \
       ~/Library/Application\ Support/scatt-companion \
       ~/Library/Logs/scatt-analyzer \
       ~/Library/Preferences/com.scatt-companion.app.plist \
       ~/Library/Preferences/com.scatt-companion.app.SCATT-Prone-Analyzer.plist
  • アプリ本体: /Applications/SCATT Companion.app
  • 心拍・記録 DB: ~/Library/Application Support/scatt-companion
  • ログ: ~/Library/Logs/scatt-analyzer
  • 環境設定: ~/Library/Preferences/com.scatt-companion.app*.plist

SCATT Expert 本体の storage.dat は SCATT Companion の管理外なので、削除しても SCATT Expert の射撃データは消えません。

Windows 10+

Windows (NSIS)

約 35 MB / per-user、管理者権限不要

初回起動 (4 ステップ)

  1. 01

    インストーラをダウンロード

    ダウンロード v0.4.12

    約 35 MB / GitHub Releases から配信

  2. 02

    .exe をダブルクリック

    ダウンロードフォルダの SCATT-Companion-Setup-0.4.12.exe をダブルクリックで実行。

  3. 03

    SmartScreen の警告を進める

    青い画面で「Windows によって PC が保護されました」と表示されたら、
    詳細情報 をクリック → 実行 ボタンを押す。

    ※ そのまま「実行しない」を押すと進めなくなります。

  4. 04

    SCATT Companion を起動

    インストール完了後、スタートメニュー (画面左下の Windows マーク) から SCATT Companion を探してクリック。

    ※ 2 回目以降は普通のアプリと同じように起動できます。

コードサイニング証明書なし配布のため、SmartScreen の初回学習を一度だけ通します。管理者権限は不要で、ユーザーフォルダ内にのみインストールされます。

削除Windows から SCATT Companion を消す +

アプリだけ削除する

設定 → アプリ → インストールされているアプリ から SCATT Companion を選び、アンインストール。

※ スタートメニュー → SCATT Companion → Uninstall でも同じ。心拍データや設定は残ります。

設定・心拍データも含めて完全に削除する

スタートメニューで「cmd」と入力 → コマンドプロンプト を起動。下のコマンドを貼り付け (右クリック貼り付け) → Enter。 

> rmdir /s /q "%APPDATA%\scatt-companion" "%LOCALAPPDATA%\scatt-analyzer"
reg delete "HKCU\Software\scatt-companion" /f
  • 心拍・記録 DB: %APPDATA%\scatt-companion
  • ログ: %LOCALAPPDATA%\scatt-analyzer
  • 環境設定 (レジストリ): HKCU\Software\scatt-companion

※ アプリ本体は先にアンインストーラで消してから実行してください。

SCATT Expert 本体は別ソフトです。SCATT Companion を削除しても SCATT Expert の射撃データは消えません。

よくある質問

SCATT Electronics 公式ですか? +

違います。Apache License 2.0 の独立プロジェクトで、SCATT が出力する storage.dat を読みに行きます。

Windows でも動きますか? +

v0.4.12 から NSIS インストーラを配布。per-user、管理者権限不要。動作検証は macOS 主体です。

射撃種目は? +

50m ライフルと 10m エアライフルに対応。ターゲット幾何と 10a / 10b / 9c の判定半径が ISSF 仕様に追従します。

心拍計はどうやって使うの? +

Apple Watch + HeartCast、または Polar H10 等の胸ベルトから BLE Heart Rate Profile を受信します。

データを別 PC に移せますか? +

本ソフトの補助 DB は ~/Library/Application Support/scatt-companion/ をコピーで移行できます。SCATT 本体の storage.dat は SCATT 本体側で。

クラウドにデータを送りますか? +

送りません。すべての処理は手元の Mac / PC で完結します。
