観察できる指標

SCATT 互換指標

10a / 10a-0.5 / 10b / 9c / S1 を本家と同じ命名で。

本家にない補助指標

重心 R95、撃発時速度、力み (8-12Hz)、フォロースルー一貫性。

z-score ランキング

過去 shot との偏差を悪い順に並べる。

手のふるえと呼吸

pre-trigger を周波数別に分解 (呼吸 0.15-0.5Hz / 手ふるえ 8-12Hz)。

反動の詳細

peak 振幅、settle time、フォロースルー R95、反動方向の円形 σ。

心拍 / HRV

Apple Watch (HeartCast) や Polar H10 等を BLE で受信。

自然言語フィードバック

LLM なしで動作。所見をローカルで生成。

セッション横断

前半 vs 後半、ベスト vs ワースト、Series 10 発ブロック表示。

ローカル完結

クラウド送信なし、API キー不要。

3 つのモード

50m スポーツライフル

伏射 (Prone)

SCATT 互換 + 反動詳細 + 心拍。

立射

AR (10m エアライフル)

撃発時速度・S1・フォロースルーの一貫性。

ターゲット非依存

ホールド練習

銃の静止度だけを見る。重心 R95 と力み中心。

よくある質問

SCATT Electronics 公式ですか? +

違います。Apache License 2.0 の独立プロジェクトで、SCATT が出力する storage.dat を読みに行きます。

Windows でも動きますか? +

v0.4.12 から NSIS インストーラを配布。per-user、管理者権限不要。動作検証は macOS 主体です。

射撃種目は? +

50m ライフルと 10m エアライフルに対応。ターゲット幾何と 10a / 10b / 9c の判定半径が ISSF 仕様に追従します。

心拍計はどうやって使うの? +

Apple Watch + HeartCast、または Polar H10 等の胸ベルトから BLE Heart Rate Profile を受信します。

データを別 PC に移せますか? +

本ソフトの補助 DB は ~/Library/Application Support/scatt-companion/ をコピーで移行できます。SCATT 本体の storage.dat は SCATT 本体側で。

クラウドにデータを送りますか? +

送りません。すべての処理は手元の Mac / PC で完結します。
