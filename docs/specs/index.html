1. 概要

SCATT Companion は、SCATT Electronics 社の射撃トレーニングソフト SCATT Expert が保存する練習データを独自リバースで読み解き、 本家にない補助分析を提供するデスクトップアプリ (macOS / Windows)。

提供する付加価値:

  • 本家にない 32 個の補助指標 (重心 R95 / 力み / フォロースルー比 等)
  • セッション横断レビュー (前半 vs 後半 / ベスト vs ワースト)
  • 個人 z-score ランキング
  • BLE 心拍計 (Apple Watch / Polar H10 等) 連携
  • ローカル自然言語フィードバック (オフライン)
  • 軌跡アニメーション再生 / セッション PDF 出力

3. 3 つの練習モード

ホーム画面でモードを選ぶと、Dashboard の KPI / グラフ / 表示要素が一括最適化される。

伏射 (Prone)

推奨種目50m ライフル (5.6mm)
主役 KPI10a / 10a-0.5 / S1 / R95-0.5s
特徴SCATT 互換指標中心。反動詳細・心拍 / HRV を活用。

AR (10m エアライフル)

推奨種目10m AR (4.5mm)
主役 KPI撃発時速度 / S1 / 撃発時心拍 / フォロースルー安定
特徴立射特化。フォロースルーの一貫性を重視。

ホールド練習 (ターゲット非依存)

主役 KPI撃発時速度 / S1 / 重心 R95-0.5s / 力み (8-12Hz)
隠す指標10a / 10a-0.5 / 10b / 9c / R95 系 (ターゲット中心からの距離指標)
特徴黒点を狙わず、銃の静止度だけを評価。基礎練向き。

6. グラフ 30 種

軌跡・座標

  • 速度時系列 (発射 = 0)
  • 現 trace の軌跡 (X-Y)
  • 発射前後 軌跡 重ね (青=前 / 赤=後 / 黄=発射点)
  • ホールド軌跡 (重心中心)
  • 銃の傾き 時系列

セッション傾向

  • 安定度 推移 (S2 / S1 / R95-2s)
  • 10a / 10a-0.5 推移 / S1 推移 (10 発移動平均線付き)
  • 撃発時 銃の傾き / 撃発タイミング / 静止時間 推移
  • 反動 peak amplitude / settle time 推移

相関 (AR で実証済み)

  • S1 vs 着弾点距離 (r=+0.38)
  • 重心 R95 vs 10a-0.5 (r=-0.37)
  • S1 vs ホールド時間 (r=-0.33)
  • 心拍 vs S2 / HRV vs S2 / 撃発タイミング vs S2

診断・反動

  • FFT スペクトル (X / Y 別、呼吸/心拍/力み帯ハイライト)
  • 反動軌跡 オーバーレイ / 反動方向 × 振幅 / 発射後速度
  • ベスト 5 vs ワースト 5 比較

各グラフは 右クリック → 計算式 で式と意味を確認できる。

7. データ管理

読み取り元

  • SCATT 本家: ~/Library/Application Support/SCATT Electronics/Scatt Expert/storage.dat (read-only)
  • persons / sessions / shots / traces テーブル
  • trace.data BLOB は XOR + qCompress + QDataStream を独自リバースで復号

本ソフトが書く先

  • 補助 DB: ~/Library/Application Support/scatt-companion/extra.db (心拍 / HRV、shot 除外フラグ)
  • profiles ディレクトリ: 射手別 extra.db
  • ログ: ~/Library/Logs/scatt-analyzer/app.log
  • QSettings: ~/Library/Preferences/com.scatt-companion.app.plist

バックアップ / 復元

Settings から バックアップ (.zip) / 復元。zip 内に extra.db + 設定 plist + manifest。

shot 除外 / 削除

  • 除外 (hidden): 補助 DB に hidden=1。SCATT 側は触らない。
  • 異常 shot 自動削除: 中心から 200mm 以上は SCATT 本家の shots.deleted=1 も立てる (要承認)。

9. プライバシー

  • すべての処理は手元の Mac / PC で完結。クラウド送信なし、API キー不要。
  • ローカル自然言語フィードバックは LLM を使わず、ルールベース。
  • BLE 心拍は受信のみ、外部送信なし。
  • 更新確認は任意 (Settings で URL 指定時のみ)。scatt-companion/{version} という User-Agent で公開 manifest URL のみ叩く。
