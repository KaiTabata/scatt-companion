1. 概要

SCATT Companion は、SCATT Electronics 社の射撃トレーニングソフト SCATT Expert が保存する練習データを独自リバースで読み解き、 本家にない補助分析を提供する macOS デスクトップアプリです。

本ソフトが提供する付加価値:

  • 本家にない 32 個の補助指標 (重心 R95 / 力み / フォロースルー比 等)
  • セッション横断レビュー (前半 vs 後半 / ベスト vs ワースト)
  • 個人 z-score ランキング (「今回特にダメな指標」を浮かび上がらせる)
  • BLE 心拍計 (Apple Watch / Polar H10 等) 連携
  • ローカル自然言語フィードバック (オフライン、API 不要)
  • 実 AR データから検証された相関グラフ 5 種
  • 軌跡アニメーション再生
  • セッション PDF 出力

3. 3 つの練習モード

ホーム画面で選択するモードによって、Dashboard の KPI / グラフ / 表示要素が一括最適化されます。

伏射 (Prone)

項目内容
推奨種目50m ライフル (5.6mm)
主役 KPI 4 枚10a / 10a-0.5 / S1 / R95-0.5s
デフォルトグラフvelocity / scatter / 安定度推移 / 直近5発 / cant 推移 / spectrum / trace_xy / 撃発タイミング推移 / hold 推移
特徴SCATT 互換指標中心。反動詳細・心拍 / HRV を活用。

AR (10m エアライフル)

項目内容
推奨種目10m AR (4.5mm)
主役 KPI 4 枚撃発時速度 / S1 / 撃発時心拍 / フォロースルー安定 (post-0.5s R95)
デフォルトグラフvelocity / 発射前後 軌跡 重ね / S1 vs 着弾距離 / spectrum / 重心 R95 vs 10a-0.5 / S1 推移 / hr_time / trace_xy / cant 推移
大型撃発時速度Dashboard 上部に巨大表示 (信号色)
特徴立射特化。反動振幅でなく フォロースルーの一貫性 を重視。実 AR データから検証された相関グラフを中心配置。

ホールド練習 (ターゲット非依存)

項目内容
推奨種目10m AR
主役 KPI 4 枚撃発時速度 / S1 / 重心 R95-0.5s / 力み (8-12Hz)
デフォルトグラフvelocity / ホールド軌跡 (重心中心) / spectrum / cant 推移 / cant 時系列 / 撃発タイミング推移 / hold 推移 / cant 揺れ推移 / trace_xy
隠す指標10a / 10a-0.5 / 10b / 10b-0.5 / 9c / R95 系すべて (ターゲット中心からの距離指標)
ミニターゲット非表示
特徴黒点を狙わず、銃そのものの静止度だけを評価。プレッシャーなしの基礎練に最適。

6. グラフ 30 種

主な系統別に分類:

軌跡・座標

  • 速度時系列 (発射 = 0)
  • 現 trace の軌跡 (X-Y)
  • 発射前後 軌跡 重ね (青=前 / 赤=後 / 黄=発射点、重心ズレ表示)
  • ホールド軌跡 (重心中心) — ターゲット非依存
  • 銃の傾き 時系列

セッション傾向

  • 安定度 推移 (S2 / S1 / R95-2s)
  • 直近 5 発の S2 棒
  • 発射点 散布図
  • 10a / 10a-0.5 推移
  • 撃発時 銃の傾き 推移
  • 撃発タイミング 推移
  • 静止時間 推移
  • 反動 peak amplitude / settle time 推移
  • 発射前 銃傾きの揺れ 推移
  • S1 推移 (10 発移動平均線付き)

相関 (AR で実証済み)

  • S1 vs 着弾点距離 (r=+0.38、速度↓で命中↑)
  • 重心 R95 vs 10a-0.5 (r=-0.37、銃の安定度と精度)
  • S1 vs ホールド時間 (r=-0.33)
  • 発射前後 R95 比 vs 10a-0.5
  • 心拍 vs S2 / HRV vs S2
  • 撃発タイミング vs S2
  • コンディションマップ (心拍 × HRV × S2)

診断・反動

  • FFT スペクトル (X / Y 別) — 速度ベース、呼吸/心拍/力み帯ハイライト + 「数式」ダイアログ
  • 反動軌跡 オーバーレイ
  • 反動方向 × 振幅
  • 発射後速度
  • ベスト 5 vs ワースト 5 比較

各グラフは 右クリック → 計算式 でその式と意味を確認可能。

7. データ管理

読み取り元

  • SCATT 本家: ~/Library/Application Support/SCATT Electronics/Scatt Expert/storage.dat (read-only)
  • persons / sessions / shots / traces テーブル
  • trace.data BLOB は XOR + qCompress + QDataStream を独自リバースで復号 (詳細)

本ソフトが書く先

  • 補助 DB: ~/Library/Application Support/scatt-companion/extra.db
    • 心拍 / HRV (BLE 受信)
    • shot ごとの除外フラグ (空撃ち等)
    • note / IMU (将来)
  • profiles ディレクトリ: 射手別 extra.db (複数 profile 切替対応)
  • ログ: ~/Library/Logs/scatt-analyzer/app.log
  • QSettings: ~/Library/Preferences/com.scatt-companion.app.plist

バックアップ / 復元

Settings から バックアップ (.zip 書き出し) / 復元 を実行可。zip 内に extra.db + 設定 plist + manifest。

shot 除外 / 削除

  • 除外 (hidden): shot 単位、補助 DB に hidden=1 を立てる。SCATT 側は触らない。集計から除外。
  • 異常 shot 自動削除: 中心から 200mm 以上 (Settings で変更可) は SCATT 本家の shots.deleted=1 も立てる (物理削除に近い、要承認ダイアログ)。

9. プライバシー

  • すべての処理は手元の Mac で完結。クラウド送信なし、API キー不要。
  • ローカル自然言語フィードバックは LLM を使わず、ルールベース。
  • BLE 心拍は受信のみ、外部送信なし。
  • 更新確認は任意 (Settings で URL 指定時のみ)。確認時は scatt-companion/{version} という User-Agent で公開 manifest URL のみ叩く。
